Instrucciones para realizar la instalación del Programa.

1. Se debe contar con el programa DEV-C++ 4.9.9.2.
2. Debe clonar el repositorio en su computador.
3. Debe abrir el archivo MMMM.c en el Dev-C++.
4. Debe configurar en Execute, Parameters el archivo que contiene la información de las cartas.
5. Debe seleccionar la opción Compilar y Correr.
